# Apuntes de Markdown

Markdown es un lenguaje de marcado ligero que permite dar formato al texto de una forma sencilla y rápida.  
Se usa principalmente en archivos README.md, documentación, blogs y foros.  
Su objetivo es hacer que el texto sea fácil de escribir y de leer, sin necesidad de etiquetas complicadas como HTML.


## Encabezados

Los encabezados se usan para crear títulos y subtítulos.  
Se definen con el símbolo `#`. Cuantos más `#` uses, más pequeño será el título.
Ejemplo:
``` readme
# Título 1
## Título 2
### Título 3 
#### Título 4
```
Muestra:
# Título 1  
## Título 2  
### Título 3  
#### Título 4

--------------
## Sintaxis general: 

Sirven para aplicar formato visual al texto.  
Se usan asteriscos o guiones bajos para cursiva y negrita.

Ejemplo:
```readme
Esto está en __negrita__
Esto está en **negrita** también 

Esto está en _cursiva_
Esto está en *cursiva*

Y ambos es __*TEXTO*__
```
